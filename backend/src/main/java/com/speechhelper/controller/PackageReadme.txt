Controller Package
-This package contains controller classes, which contain our endpoints to interface with the front end.
-The ErrorController currently has only a default endpoint set for the backend to display a message that it is running
-The SpeakingHelperController contains all of our main SpeakingHelper commands, such as creating speech objects, running analysis, and getting feedback returned.
