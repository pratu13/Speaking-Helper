Utilities Package
-Contains static singleton classes with helpful utilities throughout the program.
-FileDownloadUtility parses a downloaded file for its text and returns it, as well as finds a file by URL.
-FillerWordsUtility parses the transcribed speech text and returns feedback on the amount of filler words in relation to total speech.
-SpeechRateUtility finds the words per minute that a person is speaking.
-WordCountUtility analyzes the transcribed text and returns a breakdown of how often any given word is being used.
